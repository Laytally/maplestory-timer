# 메이플스토리 타이머 (제작자 : Laytally)

메이플스토리 설치기 타이밍을 맞추기 위한 간단한 알람 프로그램입니다.
지정된 시간이 지나면 알람 소리가 반복 재생됩니다.

## 디스코드
https://discord.gg/6ysVFJ6umJ

## 실행 방법
Python 3.11.9 버전과 vscode를 설치하시고 
vscode를 관리자 권한으로 실행하시면 됩니다.

## 기능

* 지정된 시간 타이머
* 알람 소리 반복 재생
* 키보드 입력으로 알람 제어
* 멀티스레드를 이용한 타이머 실행

## 사용 방법

1. 필요한 라이브러리 설치

```
pip install pygame pynput
```

2. 프로젝트 폴더에 알람 소리 파일 추가

```
sound.mp3
```

3. 프로그램 실행

```
python py.py
```

## 조작 키

| 키 | 기능             |
| - | -------------- |
| Z | 타이머 시작 / 알람 정지 |

## 타이머 시간 변경

코드에서 아래 값을 수정하면 됩니다.

```
타이머_시간 = 54
```

설치기 시간에 맞게 초 단위를 조정하세요.

## 파일 구조

```
project/
 ├ py.py
 ├ sound.mp3
 └ README.md
```

## 주의사항

* 프로그램 실행 중에는 `sound.mp3` 파일이 같은 폴더에 있어야 합니다.
* 게임 매크로가 아닌 **단순 알람 프로그램**입니다.
